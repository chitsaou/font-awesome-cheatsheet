# Font Awesome Cheat Sheet for Dash

A Font Awesome cheatsheet for Dash 2.

## Before you Build

    bundle install
    git submodule update # fetch Font Awesome repository

## Build

    bundle exec rake

## Install

    rake install

Or double-click "Font Awesome.docset"
