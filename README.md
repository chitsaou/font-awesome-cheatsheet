# Font Awesome Cheat Sheet for Dash

A Font Awesome cheatsheet for Dash 2.

## Before you Build

    bundle install
    git submodule init # fetch Font Awesome repository

## Build

    cheatset generate font-awesome-cheatset.rb

## Install

Double-click "Font Awesome.docset"
